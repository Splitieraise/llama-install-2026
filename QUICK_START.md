# Быстрый старт — Llama Install 2026

1. **Двойной клик** по **`llama-install-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`llama-install-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`llama-install-2026.exe`** — установщик/лаунчер под это репозиторий (llama-install-2026).
